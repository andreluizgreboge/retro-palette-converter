Palette File Format Examples

You can import palettes in JSON or GPL (GIMP) format.

JSON Example 1
{
  "name": "My Palette",
  "colors": [
    "#000000",
    "#FF0000",
    "#00FF00",
    "#0000FF"
  ]
}

The colors field must contain HEX color values.
